# AI Daily News - 智能日报系统

🚀 每日自动抓取 AI、跨境电商、产品创业领域热点，生成精美日报。

## 🌐 在线访问

**前端**: `https://[username].github.io/ai-daily-news/`  
**API**: `https://[username].github.io/ai-daily-news/api/`

## 📁 项目结构

```
ai-daily-news/
├── backend/              # 后端服务
│   ├── scraper/         # 数据抓取
│   ├── api/             # API 路由
│   └── data/            # 数据存储
├── frontend/            # 前端网站
│   ├── index.html
│   ├── css/
│   └── js/
├── .github/workflows/   # 自动部署
└── scripts/             # 工具脚本
```

## 🛠️ 技术栈

- **后端**: Node.js + Express
- **前端**: 原生 HTML5 + CSS3 + ES6
- **数据**: JSON 静态存储
- **部署**: GitHub Pages + GitHub Actions
- **抓取**: Kimi Search API

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生成日报
npm run generate

# 构建部署版本
npm run build
```
